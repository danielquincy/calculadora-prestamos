# Calculadora de Préstamos

**Calculadora de Préstamos** es una aplicación web que permite calcular de forma sencilla y rápida el monto de pagos, intereses y cronogramas de amortización para préstamos personales.

## Características principales

- Calculadora interactiva de préstamos directamente en el navegador.
- Permite ingresar monto, tasa de interés, plazo y otros parámetros del préstamo.
- Genera el cronograma de pagos y calcula el interés total.
- Visualización clara y fácil de usar.
- Sin necesidad de instalar nada: completamente web.

## Estructura del repositorio

```
calculadora-prestamos/
├── index.html      # Página principal y único archivo de la calculadora
```

## Uso

1. Abre el archivo `index.html` en tu navegador preferido.
2. Ingresa los datos solicitados (monto, tasa de interés, plazo, etc.).
3. Presiona el botón de cálculo para ver tus resultados.
